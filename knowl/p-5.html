  1. Navigate to the SageMath website 1 
  2. Click on Sage on CoCalc 2 
  3. Create a CoCalc account 3 
  4. Go to Your Projects 4  on CoCalc and create a new project.
  5. Start your new project and create a new worksheet. Choose the SageMath Worksheet option.
  6. Enter SageMath code into the worksheet and run it by clicking Run or using the shortcut Shift + Enter.
  7. Save your worksheet as a PDF for your records.
  8. To learn more about SageMath worksheets, refer to the official documentation 5
